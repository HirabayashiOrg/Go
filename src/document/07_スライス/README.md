# スライス
### 基本
Goでは配列よりもスライスを使う  
配列がデータそのものを持つのに対して、  
スライスはポインタを保持している  
```
array := [...]int{1,2,3,4,5}  // 配列
slice := array[開始位置:終了位置]
```
> 開始位置から「終了位置-開始位置」個の要素数を切り出す  
> 開始位置、終了位置は省略可能  

### 要素の個数
```
a := []int{1,2,3,4,5}
b := a[:]
len(b)  // 5
```
### スライスの最大容量
```
a := []int{1,2,3,4,5}
b := a[:3]
cap(b)  // 5
```
### Sliceの生成
```
s := []int{1,2,3,4,5}
```
> []内に何も書かないとスライスを生成する  
> 配列の場合は[...]  
```
s := make([]int, 要素数)
```
> makeを使った生成
>   第一引数：生成するデータ型を定義
>   第二引数：要素数
### 要素の追加
```
s := []int{1,2,3}
s = append(s, 4, 5)   // [1,2,3,4,5]
```
### スライスのコピー
```
s1 := []int{1,2,3}
s2 := make([]int, 3)
n  := copy(s2, s1)
```
> copyの使い方
>   第一引数：コピー先のスライス
>   第二引数：コピーしたい配列orスライス
>   戻り値：要素数
